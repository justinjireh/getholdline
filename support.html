Support

Beta support · Updated July 19, 2026

Email support@getholdline.com. The target response time for beta and paid customers is two business days. This mailbox must be activated and tested before invitations go out.

Include

  • Holdline version and release profile
  • Windows version and whether the PC uses an NVIDIA GPU
  • Microphone type and the app where output was expected
  • Exact steps and what happened instead
  • Diagnostics only if you reviewed and chose to export them

Do not include

Do not send recordings, dictated text, passwords, access tokens, payment information, or sensitive workplace data. Holdline support will never ask for your Microsoft password.

Quick checks

  • Confirm Windows microphone permission is enabled.
  • Select the intended microphone from Holdline's tray panel.
  • Try Fast accuracy mode and a text editor such as Notepad.
  • Quit and reopen Holdline after attaching a new audio device.

Security reports

For a suspected vulnerability, follow the private reporting instructions on the security page instead of posting exploit details publicly.
